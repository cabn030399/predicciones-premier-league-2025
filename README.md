# 🏆 Predicciones Premier League 2024-2025

## 🎯 Objetivo
Desarrollar un modelo de machine learning usando **Random Forest** para predecir resultados de partidos de Premier League, analizando 5 temporadas completas de datos históricos.

## 📊 Datos Utilizados
- **Fuente:** Datos históricos Premier League 
- **Período:** 2024-2025 (2 temporadas completas)
- **Variables:** Equipos, goles, estadísticas de partidos, fechas
- **Tamaño:*368* 

## 🔬 Metodología
1. **Exploración y limpieza** de datos históricos
2. **Ingeniería de características** para variables predictivas
3. **Modelado Random Forest** para clasificación de resultados
4. **Evaluación y optimización** del modelo
5. **Interpretación** de factores más influyentes

## ❓ Preguntas de Investigación
1. ¿Qué variables predicen mejor el resultado de un partido?
2. ¿Cuál es la precisión alcanzable en predicciones deportivas?
3. ¿Existen patrones temporales que mejoren las predicciones?

## 📈 Resultados Clave
- **Precisión del modelo:** 61.1%
- **Variables más importantes:**tiros a porteria y tiros en general 
- **Insights deportivos:** la localia juega un papel importante

- Predicciones informadas para aficionados
